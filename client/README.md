## Run Locally

Clone the project

```bash
  git clone https://github.com/bakliwalvaibhav1/react-navigation-v6.git
```

Go to the project directory

```bash
  cd react-navigation-v6
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
